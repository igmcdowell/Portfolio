Anmo is a mobile annotation web-application prototype built for Mobile Firefox running on the Nokia N900 handset. It allows users to highlight, star, and write notes on web pages, and offers the ability to toggle between annotate and review modes. Annotations are stored in the cloud and can be accessed, changed, and reviewed from any device. Anmo was a class final project in collaboration with Alison Meir and Jessica Voytek.

The detailed Anmo process book provides a thorough description of the research and design process, and an overview of the final result.
