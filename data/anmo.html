Anmo is a mobile annotation web-application prototype built for Mobile Firefox running on the Nokia N900 handset. It allows users to highlight, star, and write notes on web pages, and offers the ability to toggle between annotate and review modes. Annotations are stored in the cloud and can be accessed, changed, and reviewed from any device. Anmo was a class final project in collaboration with Alison Meir and Jessica Voytek.

The detailed Anmo process book provides a thorough description of the research and design process, and an overview of the final result.

The Anmo interface.

Initial planning for Anmo encompassed brainstorming, contextual inquiry and initial concept sketches.

Brainstorming

We started with the broad category of mobile browser extensions, and quickly gravitated toward the annotation space. Many of the final product features came out of this initial session.

Contextual Inquiry

To get an idea of existing study habits, we observed a cross section of students engaged in annotation work for their classes. This exercise gave us useful information on the types of annotations made, typical highlight lengths, and typical study environments. Several insights had direct impact on product features.

Concept Sketches

The final planning stage was to storyboard some typical use cases by creating low-fidelity sketches. This exercise ensured that we had fully thought through our use cases and had a workable interface.

We started with a paper prototype to usability test the basic interactions. This stop motion video shows the prototype in action. Once we'd worked out the usability issues, we built the prototype using Javascript/CSS with Python in App Engine for the back end.

Anmo Project Components (Creator in parenthesis):
  • Anmo Annotation Front End - Provides tap-based highlighting and starring (Jessica) and asynchronously transmits annotation data to the back-end via AJAX (Ian).
  • Anmo Review Front End - Collects highlights in a single view, provides controls to expand/collapse highlights. (Ian)
  • Anmo Back End - Python running on App Engine. Stores annotations and drives front end. (Ian)
